# Openingo

Openingo is na android app. Its aimed for aspiring chess players that have difficuilty with memorizing chess openings.

## Installation

Before running the project, ensure you have:

Android Studio (latest version)\
JDK 11 or higher\
Gradle installed\
An Android device or emulator\
Steps to run the project:\
Clone the repository\
bash\
Copy\
Edit\
```bash
git clone https://github.com/Martin49232/chessProject.git
```
Open Android Studio and select "Open an Existing Project"\
Select the cloned folder\
Sync Gradle and install dependencies\
Run the app on an emulator or a physical device

## Usage
You download the app and open it. Choose an opening you would like to memorize and than let the magic happen
